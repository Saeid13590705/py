# محاسبه‌گر دامنه میان‌چارکی (IQR)

این برنامه پایتون دامنه میان‌چارکی (Interquartile Range) را برای مجموعه‌ای از اعداد محاسبه می‌کند.

## ویژگی‌ها

- دریافت اعداد از کاربر
- محاسبه آمار توصیفی شامل:
  - کمینه (MIN)
  - چارک اول (Q1)
  - میانه (MED)
  - چارک سوم (Q3)
  - بیشینه (MAX)
  - دامنه میان‌چارکی (IQR)
- شناسایی داده‌های پرت (Outliers)
- نمایش نمودار شماتیک از توزیع داده‌ها
- قابلیت اجرای متعدد بدون نیاز به راه‌اندازی مجدد
